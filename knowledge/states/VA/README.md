# Virginia DCJS Regulatory Knowledge Base

This directory contains regulatory documentation for Virginia Department of Criminal Justice Services (DCJS) security guard licensing.

## Documents to Add

1. **DCJS_Private_Security_Regulations.pdf** - Core regulatory framework
2. **Armed_Guard_Requirements.pdf** - Armed security officer licensing requirements
3. **Unarmed_Guard_Requirements.pdf** - Unarmed security officer licensing requirements
4. **Training_Standards.pdf** - Required training hours and curriculum

## Sources

- [DCJS Private Security Services](https://www.dcjs.virginia.gov/licensure-and-regulatory-affairs/private-security-services)
- [DCJS Regulations (6VAC20-171)](https://law.lis.virginia.gov/admincode/title6/agency20/chapter171/)

## License Types

| Type | Initial Training | Renewal Training | Renewal Period |
|------|-----------------|------------------|----------------|
| Armed Security Officer | 40 hours | 8 hours | 2 years |
| Unarmed Security Officer | 18 hours | 4 hours | 2 years |

## How to Update

1. Download latest regulation PDFs from DCJS website
2. Place in this directory with descriptive names
3. Update `metadata.json` with any changes to requirements
4. Run RAG indexing to update embeddings
